# TideTerm

[English](./README.md) | 中文

TideTerm 是一款现代化终端应用，在传统终端的基础上，提供文件预览、网页浏览、内置编辑器、AI Chat 等“图形化块（blocks）”能力。支持 macOS / Linux / Windows。

本仓库是 **Wave Terminal**（Apache-2.0，Command Line Inc.）的 fork。fork 相关的主要改动请见 `MODIFICATIONS.md`。

## 功能亮点

- 拖拽式块布局：终端、预览、Web、编辑器、AI Chat
- 远程连接（SSH/WSL），并在同一界面提供远程文件列表与预览
- 内置编辑器，便于直接编辑远程文件
- Command Blocks：将单个命令隔离到独立块中运行与监控
- `wsh` 命令行工具：控制工作区、在本地与远程之间搬运文件
- 中英文双语界面：默认英文，可即时切换（无需重启）
